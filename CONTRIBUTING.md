# Contributing

Thank you for your interest in contributing to this project.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your change.
3. Make your updates with clear, focused commits.
4. Test the script locally.
5. Open a pull request with a short description of the change.

## Pull Request Guidelines

- Keep changes focused on one topic.
- Explain why the change is useful.
- Include screenshots or terminal output when helpful.
- Update documentation when behavior changes.

## Local Testing

Run the script with:

```bash
bash simple-interest.sh
```

Check that the output matches the expected simple interest formula:

```text
principal * rate * time / 100
```

## Code Style

- Use clear variable names.
- Keep shell commands portable where possible.
- Add comments only when they clarify non-obvious logic.
