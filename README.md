# Simple Interest Calculator

This repository contains a Bash script that calculates simple interest for a given principal amount, annual rate of interest, and time period.

Simple interest is calculated with:

```text
simple interest = principal * rate * time / 100
```

## Files

- `simple-interest.sh`: Bash script for calculating simple interest
- `LICENSE`: Apache License 2.0
- `CODE_OF_CONDUCT.md`: Community participation guidelines
- `CONTRIBUTING.md`: Contribution process for this project

## Usage

Run the script from a terminal:

```bash
bash simple-interest.sh
```

Enter the requested values when prompted:

```text
Enter the principal:
1000
Enter rate of interest per year:
5
Enter time period in years:
2
```

The script displays the simple interest amount.
