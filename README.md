# Paradise Nursery Shopping Application

Paradise Nursery is a React shopping cart application for an online plant shop.

## Features

- Landing page with a branded hero section and background image
- About Us page with company details
- Product listing page with three plant categories
- Eighteen unique houseplants with thumbnails, names, prices, and descriptions
- Shopping cart powered by Redux Toolkit
- Dynamic cart icon count in the shared navigation bar
- Add to Cart, quantity increase, quantity decrease, delete, checkout, and continue shopping actions

## Tech Stack

- React
- React Router
- Redux Toolkit
- React Redux
- Vite

## Project Structure

- `src/App.jsx`
- `src/App.css`
- `src/AboutUs.jsx`
- `src/ProductList.jsx`
- `src/CartItem.jsx`
- `src/CartSlice.jsx`

## Run Locally

1. Install dependencies with `npm install`
2. Start the development server with `npm run dev`
